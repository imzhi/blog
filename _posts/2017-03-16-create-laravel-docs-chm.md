## 什么是 CHM？

CHM 文件格式是微软于 1998 年推出的基于 HTML 文件特性的帮助文件系统。被 IE 浏览器支持的JavaScript, VBScript, ActiveX, Java Applet, Flash, 常见图形文件(GIF、JPEG、PNG)、音频视频文件(MID、WAV、AVI)等等，CHM同样支持，并可以通过 URL 与 Internet 联系在一起。
